# To-Do-Lis

#### Video Demo: https://youtu.be/LJK6jPF-zy4

#### Description: In this CS50 Project, I have built a beginner level ToDo List using HTML,CSS and Javascript.

1. In this application, Users can add and keep track of tasks they plan to do.
2. Mark down the tasks they have completed
3. Delete the Tasks they want to
4. Reset the whole ToDo lists.
